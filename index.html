⌨️ KEYBOARD BASICS

키보드부터 다릅니다

맥의 키보드에는 윈도우에 없는 키들이 있어요.
가장 먼저 이 키들을 알아볼까요?

⌘

Command (⌘)

윈도우의 Ctrl 역할을 합니다. 맥에서 가장 많이 쓰는 키예요. 복사, 붙여넣기 등 거의 모든 단축키에 사용됩니다.

Ctrl → ⌘ Cmd
⌥

Option (⌥)

윈도우의 Alt 키와 비슷합니다. 특수문자 입력이나 추가 옵션을 위해 사용돼요.

Alt → ⌥ Option
⌃

Control (⌃)

맥에도 Control 키가 있지만, 윈도우의 Ctrl과 역할이 다릅니다. 주로 시스템 기능이나 터미널에서 사용됩니다.

Ctrl (일부) → ⌃ Control
🌐

Fn / Globe (🌐)

기능 키(F1~F12)를 사용하거나 이모지 입력창을 여는 데 사용돼요. 최신 맥북에서는 지구본 모양이에요.

Fn → 🌐 Fn/Globe

💡 핵심 포인트: 윈도우에서 Ctrl로 하던 단축키는 맥에서 ⌘ Cmd로 바꿔 누르면 됩니다!
예) Ctrl+C → ⌘+C

💡 PRO TIPS

알아두면 좋은 꿀팁

윈도우에서 넘어온 유저가 꼭 알아야 할 실전 팁들

1

❌ 닫기 버튼 ≠ 종료

맥에서 빨간 ✕ 버튼은 창만 닫을 뿐 앱을 종료하지 않습니다! 앱을 완전히 종료하려면 ⌘+Q를 눌러야 합니다. Dock에서 앱 아래에 점이 보이면 아직 실행 중이라는 뜻이에요.

2

🔍 Spotlight 활용하기

⌘+Space로 Spotlight를 열면 앱 실행, 파일 검색, 계산기, 단위 변환, 환율 계산까지 할 수 있어요. 윈도우의 시작 메뉴보다 훨씬 강력합니다!

3

📂 파일 경로 체계

윈도우는 C:\Users\폴더처럼 역슬래시(\)를 쓰지만, 맥은 /Users/사용자/폴더처럼 슬래시(/)를 사용합니다. 드라이브 문자(C:, D:) 대신 볼륨 마운트 개념을 사용해요.

4

⚙️ 앱 설치/삭제

설치: .dmg 파일을 열고 앱을 Applications 폴더에 드래그하면 끝!
삭제: Applications에서 앱을 휴지통으로 드래그하거나 Launchpad에서 길게 눌러 삭제해요. 별도 언인스톨러가 필요 없어요.

5

🖥️ 메뉴바는 항상 위에

윈도우에서는 각 창 상단에 메뉴가 있지만, 맥은 항상 화면 최상단에 고정됩니다. 현재 활성화된 앱의 메뉴가 표시되므로, 메뉴를 찾을 때는 항상 위를 보세요!

6

🔄 창 버튼 위치

윈도우는 우측 상단에 최소화/최대화/닫기 버튼이 있지만, 맥은 좌측 상단에 🔴🟡🟢 신호등 버튼이 있습니다. 🟢 녹색은 전체화면이에요!

❓ FAQ

자주 묻는 질문

윈도우에서 맥으로 넘어올 때 가장 많이 궁금해하는 것들

맥에서 바이러스 백신 프로그램이 필요한가요? ▼
macOS는 XProtect, Gatekeeper 등 내장 보안 기능이 강력하여 일반적인 사용에서는 별도의 백신이 꼭 필요하지 않습니다. 하지만 불법 소프트웨어를 설치하거나 보안에 민감한 업무를 한다면 추가 백신을 고려할 수 있어요.
윈도우 프로그램을 맥에서 실행할 수 있나요? ▼
직접 실행은 불가능하지만, 몇 가지 방법이 있습니다:
• Parallels Desktop: 가상환경에서 윈도우 실행 (유료, 가장 편리)
• UTM: 무료 가상머신 앱
• Wine / CrossOver: 일부 윈도우 앱을 에뮬레이션
• Boot Camp: Intel 맥에서만 가능, M칩은 지원 안 됨
맥에서 카카오톡, 한글(HWP) 등 한국 프로그램 사용 가능한가요? ▼
• 카카오톡: Mac 버전이 있습니다. App Store에서 다운로드 가능해요.
• 한글: 한컴오피스 Mac 버전이 있어요. 또는 무료 웹 버전(한컴독스)도 사용 가능합니다.
• 공인인증서: 이제 대부분의 은행/공공기관이 맥 Safari나 Chrome을 지원합니다.
• 대부분의 주요 한국 앱들은 맥 버전을 제공하거나 웹 버전으로 이용 가능해요.
맥에서 마우스 스크롤 방향이 반대인데 어떻게 바꾸나요? ▼
시스템 설정 → 마우스 → 자연스러운 스크롤 끄기로 바꿀 수 있습니다. 다만 이렇게 하면 트랙패드 스크롤 방향도 같이 바뀝니다. 마우스만 반대로 하고 싶다면 'Scroll Reverser' 같은 무료 앱을 설치하세요.
맥에서 프로그램을 어디서 다운로드하나요? ▼
• App Store: 가장 안전한 방법. 검증된 앱만 등록됩니다.
• 공식 웹사이트: Chrome, VS Code 등은 공식 사이트에서 .dmg 파일을 다운로드합니다.
• Homebrew: 개발자라면 터미널에서 brew install 앱이름으로 간편하게 설치 가능!
맥에서 한/영 전환이 잘 안 될 때는? ▼
시스템 설정 → 키보드 → 입력 소스에서 한국어(2벌식)를 추가하세요. Caps Lock을 눌러 한/영 전환이 되며, 메뉴바 우측 상단의 입력 소스 아이콘에서 현재 입력 상태를 확인할 수 있습니다. 반응이 느리다면 입력 소스를 삭제하고 다시 추가해보세요.
맥에서 마우스 오른쪽 버튼 클릭은 어떻게 하나요? ▼
여러 가지 방법이 있어요:
• 트랙패드: 두 손가락으로 탭하거나, 하단 오른쪽 모서리 클릭 (설정 필요)
• 키보드: ⌃ Control + 클릭
• 외장 마우스: 일반 마우스를 연결하면 우클릭이 바로 됩니다!

오늘의 숙제

오늘 바로 해보면 맥의 핵심 인터페이스가 훨씬 익숙해집니다

✅ 오늘 꼭 해보기

런치패드와 스포트라이트, 앱 전환을 직접 연습해보세요

런치패드와 스포트라이트를 활용해 앱을 찾고 실행해보면서, 맥에서 자주 쓰는 실행/검색/전환 흐름을 몸에 익히는 연습입니다.

1

네 손가락 오므리기로 런치패드 열기

네 손가락 오므리기로 런치패드(Launchpad)를 열고 설치된 앱 목록을 확인해보세요. 이어서 계산기와 터미널 앱을 각각 직접 실행해보세요.

2

⌘ + Space로 스포트라이트 검색하기

Command + Space로 스포트라이트(Spotlight)를 열고 "100달러"를 입력해 환율이 나오는지 확인해보세요. 그리고 계산기, 터미널을 각각 검색해 실행해보세요.

3

두 프로그램의 차이점 정리하기

방금 실행한 계산기와 터미널의 목적과 사용 방식이 어떻게 다른지 한두 문장으로 스스로 정리해보세요.

4

⌘ + Tab으로 앱 전환 연습하기
